Shortcodes for rakuten CD search

== Description ==

楽天 CD 検索です。
検索した、商品を簡単に投稿に追加する事ができる。
しかも、アフィリエイトで稼げるかもしれません。

このプラグラインを動かすには、
Rakuten product http://wordpress.org/extend/plugins/kdk-wprakuten/
をインストールする必要があります。

完全に生まれ変わった、Rakuten productを是非試してみて下さい！

バグがあれば、教えて下さい Twitter : @johna1203
もしくは、サーポートページまでお問い合わせください、http://blog.kodokuman.com/kdk-wprakuten-support/
